== Description ==


This plugin is completely custom and intended only for use on www.casterconcepts.com. It creates two post types and enhances them with meta boxes and custom fields.


== Installation ==



1. Upload to wp-content/plugins
1. Activate in Plugins


== Frequently Asked Questions ==



= What is this plugin for? =


The Caster Concepts website needs post types to properly segregate and manage data.
